## Prerequisite
1. Log in CRC.
   - Log in as developer
     ```sh
     $ oc login -u developer -p developer https://api.crc.testing:6443
     ```
   - Log in as administrator
     ```sh
     $ oc login -u kubeadmin -p (password) https://api.crc.testing:6   443
     ```
1. Create a project.
   ```sh
   $ oc project test
   ```
